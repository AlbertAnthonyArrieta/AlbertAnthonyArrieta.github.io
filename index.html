CARS Volunteer Management System

Web Application

Description

A volunteer management system created for the non-profit organization Calgary Animal Rescue Society (CARS). The purpose of the system is to allow CARS administrators to manage volunteer information, foster information, volunteer applications, foster applications, and manage events that active volunteers could register for. This project originally started as my capstone group's project for SAIT and I with a few of the others have decided to continue to work on the system for CARS after graduation. This project is still currently being worked on and you can check out the progress with the links below.

My Role

My role during this project was to work with the front-end along with 3 other group members. During the preparation stage, I lead in designing the UI and the prototype interface design to base our project on. During development I had to learn Angular and Bootstrap on my own with some help using the internet and my peers. Most of my time during development was to develop service files that communicated with the backend that uses SpringBoot RESTful api and refining the user interface using Bootstrap. Design consistency throughout the different pages in the web app was something I looked for and creating multiple forms for user input that would be communicating with the backend database that used SpringBoot.
